RESPONSE TIME STAT COLLECTOR
=

Build
-
```
npm install
npm run build
```

Run server
-
```
npm run start:server
```

Run client
-
```
npm run start:server
```
Env options

| name    | default |
|-----|---------|
| PING_INTERVAL    | 10000   |
| SERVER_TIMEOUT    | 10000   |
| MIN_RETRY_TIME    | 100        |
| MAX_RETRY_TIME    | 60000        |
| RETRY_EXP    | 2        |
| CHECK_URL    | https://fundraiseup.com/        |
